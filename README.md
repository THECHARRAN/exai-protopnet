# Adaptive Prototype Explainable AI (ProtoPNet + Clustering)

This project implements an Explainable AI classifier using:

- Prototype-based reasoning (ProtoPNet)
- Adaptive self-organizing prototypes (K-Means refinement)
- Patch-level visual explanations
- Heatmap visualization

## Setup

pip install -r requirements.txt

## Train

python train.py

## Demo

python demo.py
